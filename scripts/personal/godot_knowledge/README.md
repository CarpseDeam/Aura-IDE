# Personal Godot expertise pack

These project-authored Aura skills are deliberately outside the packaged `aura` tree. Copy the
folders beneath `skills/` into a Godot workspace's `.aura/skills/authored/` directory. Aura selects
only relevant skills from the user's request, task kind, and target files.

The pack complements `inspect_godot_api`: skills provide engineering judgment and workflow;
ClassDB provides exact version-specific signatures.
